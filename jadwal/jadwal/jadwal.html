Jadwal Kuliah

Rekayasa Web

Matakuliah ini membahas tentang rekayasa perangkat lunak,
berbagai model pengembangan perangkat lunak,
prinsip-prinsip pemandu praktik dalam rekayasa
perangkat lunak, kebutuhan perangkat lunak,
konsep perancangan, prinsip-prinsip user interface,
konsep perancangan berbasis pola, manajemen kualitas, konsep pengujian perangkat lunak.

Pemrograman Berorientasi Objek Praktik

Matakuliah ini membahas tentang environment
pemrograman java, percabangan dan perulangan, class, object, class diagram, enkapsulasi & constructor,
pewarisan (inheritance), polimorfisme (polymorphism), abstract class & interface, pemrograman berbasis GUI bag I, penanganan eksepsi, pengaksesan basis data bag I.

Sistem Cerdas

Mata kuliah ini merupakan salah satu mata kuliah minat
di bidang sistem cerdas. Mata kuliah ini bertujuan untuk memberikan pengetahuan dasar mengenai konsep Kecerdasan Buatan pada mesin yang meliputi beberap teknik dasar, yakni Searching, Planning, Reasoning, dan Learning. Sebelum mengikuti mata kuliah ini diharapkan mahasiswa telah lulus mata kuliah Pemrograman Dasar, Algoritma dan Struktur Data, serta Aljabar Linier.

Rekayasa Web Praktik

Matakuliah ini membahas tentang Pengenalan Jaringan Internet & Intranet, Konsep Teknologi Web, Server,
PHP dan Database, Konsep Web Design,
HTML, Penggunaan Tag-tag HTML, HTML Form, HTML Table, HTML List, HTML5, HTML Graphics, HTML Media, HTML APIs, CSS dan Perkembangannya, Responsive Web Desain, JavaScript Fundamental dan Penggunaannya bersama HTML, DOM (Documen Object Model),
Web Server (Apache, Nginx, IIS, Tomcat, dll)
dan konfirgurasinya, Pemrograman PHP Fundamental,
Pengenalan Database Server dan pengelolaannya,
Koneksi Database dan CRUD (Create, Read, Update, Delete),
PHP Cookies dan Session beserta Implementasinya.

Sistem Komputer

Mata kuliah ini mempelajari perbedaan definisi Arsitektur
& definisi organisasi komputer, konsep dasar sistem komputer (konsep Von Newmann), unit-unit pembentuk sistem komputer (pemroses,penyimpan dan I/O),
hubungan antar unit pembentuk sistem komputer,
cara kerja sistem komputer dan hubungan komputer
dengan dunia di luar sistem komputer. Mahasiswa akan belajar berbagai macam hal yang berhubungan dengan
komputer seperti konsep dasar system operasi,
manajemen proses, manajemen memori, penjadwalan, dan deadlock.

Metedologi Desain Perangkat Lunak

Mata kuliah ini mempelajari tentang perancangan logik,
dan perancangan fisik, dengan menggunakan alat bantu pemodelannya (data flow diagram, data dictionary,
process specification, Struktur chart, entity relationoship diagram).

English for Daily Conversation

Mata Kuliah ini mengajarkan penggunaan bahasa Inggris untuk percakapan sehari-hari serta melatih kepercayaan diri mahasiswa untuk berani berbicara dalam bahasa Inggris.

Aplikasi Automasi Perkantoran

Mata kuliah ini bertujuan menyiapkan mahasiswa untuk mengikuti ujian sertifikasi Microsoft Office Specialist
(MOS-Word). Untuk mencapai tujuan tersebut mahasiswa akan mendapatkan teori dan praktik serta diminta mengerjakan tugas sesuai dengan kisi-kisi ujian MOS Word.
